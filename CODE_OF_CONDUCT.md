# Code of Conduct

Ralph v2 follows the Contributor Covenant Code of Conduct (v2.1).

Read the full text here: https://www.contributor-covenant.org/version/2/1/code_of_conduct.html

## Enforcement

Project maintainers are responsible for clarifying standards of acceptable behavior
and will take appropriate and fair corrective action in response to any behavior
that they deem inappropriate, threatening, offensive, or harmful.

There is no guaranteed response time or SLA for conduct reports.
